html

ჰიპერტექსტის მარკირების ენა ( HTML ) არის სტანდარტული მარკირების ენა დოკუმენტებისთვის, რომლებიც შექმნილია ვებ ბრაუზერში გამოსატანად . ის განსაზღვრავს ვებ კონტენტის შინაარსს და სტრუქტურას . მას ხშირად ეხმარება ისეთი ტექნოლოგიები, როგორიცაა CSS და სკრიპტირების ენები , როგორიცაა JavaScript, პროგრამირების ენა.ვებ ბრაუზერები იღებენ HTML დოკუმენტებს ვებ სერვერიდან ან ლოკალური მეხსიერებიდან და გადააქვთ დოკუმენტები მულტიმედია ვებ გვერდებზე. HTML აღწერს ვებ გვერდის სტრუქტურას სემანტიკურად და თავდაპირველად მოიცავდა მის გარეგნობას.

html- ის შესახებ ვრცელი ინფორმაცია

css

CSS— სტილთა ფურცლების ენა, რომელიც გამოიყენება მონიშვნის ენით დაწერილი დოკუმენტის წარდგინების აღსაწერად, მაგალითად HTML-ის გამოყენებით.[1] CSS არის მსოფლიო აბლაბუდის ტექნოლოგიური ქვაკუთხედი, HTML-თან და JavaScript-თან ერთად.[2] CSS შემუშავებულია კონტენტის და მისი წარდგენის, მათ შორის ფერების, ფონტების და გაწყობის გაცალკევებისათვის.[3] ამგვარ გაცალკევებას შეუძლია გააუმჯობესოს კონტენტის ხელმისაწვდომობა, უზრუნველყოს წარდგენის თვისებების განსაზღვრის უფრო მეტი მოქნილობა და მართვადობა, უზრუნველყოს მრავალი ვებგვერდის მიერ საზიარო ფორმატირების გამოყენება შესაბამისი სტილების განცალკევებულ . css ფაილში აღწერით, რაც ამცირებს კონტენტის სტრუქტურაში სირთულესა და გამეორებადობას, ისევე როგორც უზრუნველყოს .css ფაილის კეშირება, რაც საზიარო სტილის და ფორმატირების მქონე სხვადასხვა გვერდის ჩატვირთვის სიჩქარეს აამაღლებს.

css-ის შესახებ ვრცელი ინფორმაცია

JavaScript

ფუნქციები JavaScript-ის ერთ-ერთი ფუნდამენტური სამშენებლო ბლოკია. JavaScript-ში ფუნქცია მსგავსია პროცედურისა - განცხადებების ერთობლიობა, რომელიც ასრულებს დავალებას ან ითვლის მნიშვნელობას, მაგრამ იმისათვის, რომ პროცედურა კვალიფიცირდეს ფუნქციად, მან უნდა მიიღოს გარკვეული შეყვანა და დააბრუნოს გამომავალი, სადაც აშკარა კავშირია შეყვანა და გამომავალი. ფუნქციის გამოსაყენებლად, თქვენ უნდა განსაზღვროთ ის სადმე იმ სფეროდან, საიდანაც გსურთ მისი გამოძახება.
