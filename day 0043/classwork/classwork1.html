html

HTML (HyperText Markup Language) არის ვებ-გვერდების შექმნისა და სტრუქტურირების ენა. ის გამოიყენება ვებ-გვერდის შინაარსის ორგანიზებისთვის, როგორიცაა ტექსტი, სურათები, ბმულები და სხვა ელემენტები. HTML-მა საშუალებას აძლევს ბრაუზერს, რომ სწორად აჩვენოს გვერდი.

css

CSS (Cascading Style Sheets) არის ენა, რომელიც გამოიყენება HTML დოკუმენტის სტილის განსაზღვრისთვის. CSS საშუალებას გაძლევთ ვებ-გვერდს მიანიჭოთ ფერები, შრიფტები, განლაგებები და სხვა ვიზუალური ეფექტები, რაც HTML-ს არ შეუძლია.
