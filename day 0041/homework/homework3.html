    • Frontend Development is the creation of the web user-visible part of a page or application. The main technologies are:

  • HTML (structure),

    • What is Frontend?

    Frontend refers to the part of a website or application that users interact with directly. It includes everything you see and use, such as buttons, menus, images, text, and animations. The main goal of frontend development is to create a visually appealing, responsive, and user-friendly interface.

    html

    HTML (HyperText Markup Language) is the standard language used to create the structure of a webpage. It defines how a webpage is organized and what elements it contains, such as headings, text, images, links, lists, and buttons.

    Uses of HTML:

  • To build the structure of a website.
  • To define visual elements (e.g., paragraphs, headings).
  • To embed other technologies (CSS, JavaScript).
  • To integrate links and multimedia (videos, audio).

    • history

    HTML (HyperText Markup Language) was created by Tim Berners-Lee in 1989 while he was working at CERN (European Organization for Nuclear Research). It was designed to be a simple system for sharing documents over the internet and was a key component in the development of the World Wide Web.

    css

    CSS (Cascading Style Sheets) is a language used to define the visual style of a webpage. With CSS, you can modify the appearance of HTML elements, such as colors, fonts, layout, and other design aspects.

    Main Functions of CSS:

  • Colors: Change text and background colors.
  • sizing: Set the size of elements, borders, and spacing.
  • Layout: Organize elements on the page, such as positioning menus or images.
  • Fonts: Control font style, size, and weight.
  • Responsive Design: Adjust the layout for different devices (mobile, tablet, desktop).

    • history

    CSS (Cascading Style Sheets) was developed to provide better control over the presentation and design of web pages. Prior to CSS, web designers had to rely on HTML alone to manage both structure and design, which led to limitations in styling flexibility and efficiency.

    JavaScrpit

    JavaScript is a programming language used to add interactivity and dynamic features to webpages. It enables elements on a page to respond to user actions, such as clicking buttons, submitting forms, or scrolling.

    Key Features of JavaScript:

  • Interactivity: Responding to user actions (e.g., button clicks, form submissions).
  • Dynamic Content: Changing or updating content on the page, such as refreshing lists or galleries.
  • Minimal Server Requests: Fetching data from a server and updating the page without reloading (e.g., using AJAX).
  • Multimedia Control: Managing videos and audio files.
  • Client-side Validation: Checking and validating form data to prevent errors before submission.

    history

    JavaScript was created in 1995 by Brendan Eich while working at Netscape Communications Corporation. He was tasked with developing a scripting language that could add interactivity to web pages, as websites at the time were primarily static and lacked user interaction.
